 # 📊 TECO Customer Churn Analysis

## 📌 Project Overview

This project analyzes customer churn for **TECO**, a telecommunications company, to identify key drivers of churn and provide actionable recommendations for retention. The analysis reveals that **26.5% of the customer base has churned**, posing a significant revenue risk. The insights are derived from demographic, behavioral, and service-related factors.


## 🔍 Key Findings

# 📈 Churn by Demographics
- **Senior citizens** churn at a disproportionately higher rate.
- **Gender** has no meaningful impact on churn.

### ⏳ Tenure
- Customers with **1–2 months tenure** churn at **45–50%**.
- Churn drops below **10%** after **24+ months**.
- **Early experience is critical** for retention.

### 📄 Contract Type
| Contract Type       | Churn Rate |
|---------------------|------------|
| Month-to-month      | ~55–60% of all churn cases |
| One-year            | ~11–13%    |
| Two-year            | <5%        |

### 🛠️ Add-on Services
- Without add-ons: **35–45% churn**.
- With add-ons: **<15% churn**.
- Bundling increases engagement and switching costs.

### 🌐 Internet Service Type
| Service Type        | Churn Rate |
|---------------------|-------------|
| Fiber optic         | ~40–42%     |
| DSL                 | ~19–21%     |
| No internet service | <8%         |

### 💳 Payment & Billing
- **Electronic check** users: >45% churn.
- **Credit card / automatic bank transfer**: <15% churn.
- Paperless billing alone does not indicate loyalty.

## 🧠 Recommendations

Target retention efforts on:

- ✅ New customers (low tenure)
- ✅ Month-to-month contract holders
- ✅ Fiber optic users
- ✅ Customers without bundled services
- ✅ Electronic check payers
- ✅ Senior citizens

Improve retention by promoting:

- 🔁 Longer tenure & multi-year contracts
- 📦 Value-added service bundles
- 🤖 Automated payment methods
- 🎯 Proactive early-stage engagement
